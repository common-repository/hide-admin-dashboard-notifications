Disable unnecessary dashboard notifications for admin users and boost productivity.

== Description ==

The Hide Admin Dashboard Notifications plugin enables you to disable dashboard notifications for specific or all admin users. This enables admin users to focus on more important tasks on the system which boosts productivity.


== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Hide Admin Dashboard Notifications**.
3. Install and activate the Hide Admin Dashboard Notifications plugin.

= Manual installation =

1. Upload the entire `belo-hide-admin-notifications` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Hide Admin Dashboard Notifications plugin.

= After activation =

1. Select the Hide dashboard notifications menu.
2. By default all admin users are included, select the admin to hide the notifications and click save.
